# 温馨提示 🙋

- 后端每次返回的数据不一样才会触发update，建议后端在返回的参数中加入一个唯一值字段。

- 后端返回数字一定要用字符串类型，要不会丢失精度。

- 不要声明全局变量，多个控件使用的时候会有变量污染的问题。

- 工程不兼容IE。
