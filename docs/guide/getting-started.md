# 安装与创建

支持公司内外网安装！（**内网脚手架和起步工程版本会落后外网**）

node 环境版本建议18以上！

## 安装脚手架

```bash
npm install -g kingdee-cosmic-cli
```

## 查看版本

```bash
cosmic --version
```

## 新建项目

```bash
cosmic create my-app
```

## 选择配置

- 请选择一个框架
- 是否需要 Eslint 和 Prettier ？
  
进入到新建的项目，安装对应依赖：
```bash
npm install
```

## 目前支持的框架
- React18
- Vue3
